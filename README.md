# Transit Distance Ruler

A Dash application to measure distance traveled on transit.

## Requirements:

- python 3.14 or newer

## Installation

This is tested on macOS 26. It *should* work on other operating systems, but this is not guaranteed.

```bash
python -m venv ./.venv
source ./.venv/bin/activate # Windows: source ./.venv/Scripts/activate.ps1
pip install git+https://github.com/PikalaxALT/transit-distance-ruler.git
```

## Usage

```bash
python -m transit_distance_ruler
```
